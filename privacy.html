개인정보처리방침

시행일: 2026년 4월 25일

MapleStory MVP Checker는 넥슨 결제 사용 내역을 기반으로 메이플스토리 MVP 등급을 계산하고 다음 주 예상 등급을 확인할 수 있도록 돕는 Chrome 확장 프로그램입니다.

본 확장 프로그램은 사용자의 데이터를 외부 서버로 전송하거나 별도로 저장하지 않습니다. 조회된 데이터는 사용자의 브라우저 내에서만 처리됩니다.

1. 처리하는 사용자 데이터

본 확장 프로그램은 넥슨 결제 페이지에 로그인된 상태에서 결제 사용 내역을 조회하여 메이플스토리 MVP 등급 계산에 사용합니다.

처리될 수 있는 데이터는 다음과 같습니다.

  • 결제 사용 일자
  • 결제 사용 금액
  • 게임명
  • 구매 아이템명
  • 구매 상태 정보

2. 데이터 사용 목적

조회된 데이터는 다음 목적을 위해서만 사용됩니다.

  • 최근 13주 기준 MVP 누적 금액 계산
  • 현재 예상 MVP 등급 표시
  • 다음 주 예상 등급 및 등급 하락 예상일 계산
  • 주차별 사용 금액 표시

3. 데이터 저장 및 외부 전송 여부

본 확장 프로그램은 조회한 결제 내역을 외부 서버로 전송하지 않으며, 별도의 데이터베이스나 저장소에 저장하지 않습니다.

모든 계산은 사용자의 브라우저 내에서만 수행됩니다.

4. 로그인 정보 및 인증 정보

본 확장 프로그램은 사용자의 넥슨 아이디, 비밀번호, 인증 정보, 쿠키 값을 별도로 수집하거나 저장하지 않습니다. 넥슨 결제 내역 조회는 사용자가 이미 넥슨 결제 페이지에 로그인한 상태에서 브라우저의 세션을 통해 이루어집니다.

5. 제3자 제공

본 확장 프로그램은 사용자 데이터를 제3자에게 제공하지 않습니다.

6. 참고 사항

본 확장 프로그램에서 계산되는 MVP 등급은 참고용입니다. 인벤토리로 이동하지 않은 아이템, PC방 이용 금액 등 일부 금액은 반영되지 않을 수 있으며, 실제 메이플스토리 MVP 등급과 차이가 발생할 수 있습니다.

본 확장 프로그램은 넥슨 또는 메이플스토리 공식 서비스가 아닙니다.

7. 문의

개인정보처리방침 또는 확장 프로그램 사용과 관련된 문의는 프로젝트 저장소의 이슈 페이지를 통해 남겨주세요.
