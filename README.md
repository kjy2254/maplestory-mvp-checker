# MapleStory MVP Checker

넥슨 결제 사용 내역을 기반으로 메이플스토리 MVP 등급을 계산하고, 다음 주 예상 등급과 등급 하락 예상 시점을 확인할 수 있는 Chrome 확장 프로그램입니다.

## 주요 기능

- 최근 13주 기준 메이플스토리 MVP 누적 금액 계산
- 현재 예상 MVP 등급 표시
- 다음 주 예상 등급 표시
- 등급 하락 예상일 안내
- MVP 주차별 사용 금액 정리
- 넥슨 결제 페이지 내에서 바로 확인 가능한 패널 UI 제공

## 사용 방법

1. Chrome 확장 프로그램을 설치합니다.
2. 넥슨 결제 페이지에 로그인합니다.
3. `https://payment.nexon.com/` 페이지에 접속합니다.
4. 화면 우측 하단의 **MVP 분석** 버튼을 클릭합니다.
5. 현재 예상 등급, 다음 주 예상 등급, 주차별 금액을 확인합니다.

## 참고 사항

- 본 확장 프로그램은 넥슨 결제 내역 중 **메이플스토리 사용 내역**을 기준으로 계산합니다.
- 반영되는 금액은 구매한 아이템 기준입니다.
- 인벤토리로 이동하지 않은 아이템 및 PC방 이용 금액은 반영되지 않을 수 있습니다.
- 실제 MVP 등급과 차이가 발생할 수 있으므로 참고용으로 사용해 주세요.
- 본 확장 프로그램은 넥슨 또는 메이플스토리 공식 서비스가 아닙니다.

## 개인정보 보호

본 확장 프로그램은 사용자의 결제 내역을 브라우저 내에서만 조회하고 계산에 사용합니다.

- 로그인 정보는 수집하거나 저장하지 않습니다.
- 결제 내역은 외부 서버로 전송하지 않습니다.
- 사용자 데이터를 별도로 저장하지 않습니다.

자세한 내용은 [개인정보처리방침](https://kjy2254.github.io/maplestory-mvp-checker/privacy.html)을 확인해 주세요.

## 권한 사용 안내

이 확장 프로그램은 다음 목적을 위해 최소한의 권한만 사용합니다.

- `https://payment.nexon.com/*`  
  넥슨 결제 페이지에서 확장 프로그램 UI를 표시하기 위해 사용합니다.

- `https://public.api.nexon.com/*`  
  넥슨 결제 사용 내역을 조회하여 MVP 등급을 계산하기 위해 사용합니다.

조회된 데이터는 사용자 브라우저 내에서만 처리되며 외부로 전송되지 않습니다.
